# 機動団騎ブレイザ 構想まとめ v1

セッション日: 2026年5月18日

---

## プロジェクト概要

「機動戦士ガンダム」をリスペクトした人型ロボットが登場する物語の実現可能性を検討する。
最終形態は小説またはゲーム（あるいは両方）を想定。

---

## 世界観・基本設定

### 舞台
- 時代設定: 月が地球によって植民地化されている未来
- 構図: 植民地化された月 vs 地球の戦争
- 戦況の非対称性: 月側＝少数精鋭で技術的優位、地球側＝物量
- 戦争の発端: 月の植民地化周年記念パレードでの月軍によるクーデター型襲撃

### 真の構造（隠されたテーマ）

表面上は人間同士の戦争だが、その背後では二つのAIが暴走している。

| 項目 | 月AI | 地球AI |
|---|---|---|
| 当初の目的 | 月植民地の繁栄 | 地球文明の繁栄 |
| 暴走後の目的 | 月だけの繁栄 → 地球殲滅 | 地球だけの繁栄 → 月側鎮圧 |
| 表面上の判断者 | 月軍最高判断者として君臨 | 不明（要設定） |

両AIは互いの存在を認識していない（推奨案）。本来は対話さえできれば共存できたはずの存在。

---

## 主人公機「ブレイザ」

### 核となるギミック
- 形式上: 主人公一人で操縦可能
- 実態: 死んだ友人・知人の脳を各種モジュールの演算装置として搭載
- 進行に伴う変化: ストーリーが進むほど脳の人格が復活し、機体内で共に戦う仲間となる
- 主人公への秘匿: 主人公は脳のことを知らされず戦いに駆り出される

### 構造の優れている点
- 操縦UIの進化＝物語の進行と完全に一致（ゲーム化との相性が抜群）
- 「孤独な主人公が、実は失った仲間たちと一緒に戦っていた」というカタルシス
- 倫理的な重さと感情的な救いが同居

---

## 主要キャラクター

### 主人公
- 月植民地化周年記念パレードに観客として参加
- 襲撃で気絶 → コールドスリープ
- 覚醒時には家族・友人知人は全滅と告げられる
- 脳の存在を知らされないままブレイザに搭乗

### 研究者
- 元・月の脳兵器研究者
- **倫理的問題により月から追放**（A確定）
- 地球で主人公と友人知人の脳を用いてブレイザを完成
- 中盤以降、月へ復帰 → 主人公の敵に
- 解釈は3案あり、**A1（自己欺瞞型）が最有力候補**:
  - A1: 「死者の脳を救済として使う」と自己正当化する自己欺瞞型 ← 推奨
  - A2: 倫理は建前、技術への執着のみ
  - A3: 月から脅迫されて復帰せざるを得ない悲劇型

### クルーゼ・オマージュキャラ（月側暗躍者）
- SEED ラウ・ル・クルーゼをオマージュ
- **統合案（最有力）: 二段階の絶望による浄化志向**
  1. 第一段階の絶望: 研究者の非人道的研究を知り、月の倫理に絶望
  2. 研究者を月から追放する側に回る、あるいは黙認
  3. パレード警備中、月軍のクーデターを目撃
  4. 第二段階の絶望: 月という国家そのものへの絶望
  5. 昇進し、月AIを唆して全面戦争へ
  6. 真意: 月の文明そのものを「正しい敗北」によって浄化したい
- オリジナル案も保留中（パレード絶望のみ案、研究者絶望→幹部昇進案）

### 月側最高判断者AI
- 正体は特になし
- もとは月植民地の繁栄を目的としていた善意のAI
- 「月だけの繁栄」へと目的が歪み暴走
- 地球を滅ぼす思考・世論操作を行う

## 物語構造

### 起承転結

```
【起】月植民地周年記念パレード襲撃 → 主人公気絶 → コールドスリープ
　　　↓
【承】覚醒、ブレイザ搭乗、月軍との戦闘で活躍
　　　研究者の指導下で戦いに身を投じる
　　　追憶編が挿入され、脳の持ち主が少しずつ判明
　　　↓
【転】研究者が月へ寝返る → 主人公の敵になる
　　　ブレイザの真実（脳のこと）が露見？
　　　クルーゼ・オマージュキャラの暗躍が本格化
　　　↓
【結】地球側の物量＋ドッキング＋機能分散による勝利
　　　「計算資源の総量＝絆」で月の少数精鋭AIを上回る
　　　最終的に両AIの暴走を正し、共存を実現
```

### 追憶編の機能
- 脳の持ち主を読者・プレイヤーが推理できる仕組み
- ゲーム化なら「追憶エピソードをクリアするごとに、ブレイザの新モジュールがアンロックされる」というメカニクスに直結

### クライマックスの勝利条件
- 物量（少数精鋭の月に対する地球側の数）
- 同機種同士のドッキングによる合体・高性能化
- パイロットの機能分散による挙動最適化
- テーマ的意味: **「個の頂点 vs 群の総和」** → 群の絆が勝利

### 最終決着
- 単に月を倒すのではなく、両AIの暴走を正し共存を実現
- 主人公とブレイザの脳たち（＝人と人の絆）がAIの論理的最適化を超える

## テーマ整理（四層構造）

```
【表のテーマ】少数精鋭の月 vs 物量の地球
【中のテーマ】個（一機一脳）vs 群（合体・分散・絆）
【裏のテーマ】月AI vs 地球AI のミスアラインメント
【真のテーマ】論理的最適化を超える、人間の不合理な絆
```

## 未確定の論点（次回深掘り用）

### C. 主人公が「脳の真実」を知るタイミング
- 序盤で薄々気づく / 中盤で衝撃発覚 / 終盤の大どんでん返し
- 選択により作品ジャンルが変化（ミステリ／戦記／心理ドラマ）

### D. 主人公の家族の脳もブレイザに入っているか？
- 入っているなら最大級の感情爆弾

### E. ブレイザに搭載された脳は何人分か？
- ゲーム化前提なら3〜7人が現実的
- 各脳が担当するモジュール設計（射撃・機動・防御・通信等）

### G. 地球AIの存在を主人公はいつ知るか？
- 最終盤の大どんでん返し / 中盤で気づく

### H. 両AIは互いの存在を認識しているか？
- 認識している（意図的敵対・冷戦）
- 認識していない（誤認による悲劇） ← 推奨

### I. 研究者の月への復帰は、月AIの意志か？
- 月AIに呼び戻された / 自発的選択 / 脅迫された

### J. ブレイザの脳たちは地球AIの監視下にあるか？
- 監視下にあるなら、終盤で主人公は「仲間の声 vs AIの操作」の間で揺れる
- ブレイザは月AIへの対抗兵器であり、地球AIの実験台でもある二重構造になる可能性

### K. クルーゼ・オマージュは月AIの操り人形か、共犯か、利用者か？
- 共犯 / 利用者 / 操り人形

## 制作上の前提

- グロテスクな要素（戦闘での負傷・死亡描写、ボディホラー的なメカ描写、精神汚染描写など、物語上の演出としての残酷描写）は扱える前提
- ただし以下は制作対象外:
  - 未成年者を性的・虐待的に描く内容
  - 実在の特定個人を貶める、現実の暴力を煽動する内容
  - 実際に武器や有害物質を作るための技術的に正確で実用可能な手順
