# gts-scale-descriptions反映 改稿方針

## 目的

Claude作成の `C:\Users\ikasi\.claude\skills\gts-scale-descriptions` を、そのまま移植するのではなく、`百川十々瀬の巨大化記録` の既存canonに合う形へ変換して、今後の改稿の判断基準にする。

本方針では特に以下を重視する。

- スケール帯ごとの比較対象の更新
- 十々瀬視点と観測側視点の落差
- 手指以外の破壊ベクトルの増強
- 雲・大気・海洋・軌道の扱いの現実寄り修正
- 宇宙編での km 桁演出の強化

## 結論

外部スキルは「段階別の語彙棚」としては有用。
ただし、そのままだと本シリーズには合わない癖がかなりある。

採用する中心要素は以下。

1. スケール帯ごとの語彙切替
2. 上下視点の落差
3. 破壊を身体感覚の延長として書くこと
4. 数値そのものを演出にすること

逆に、そのまま採らない要素は以下。

1. 雲が腰・肩・胸の周辺にあるような記述
2. 夫・帰還・恋愛中心の心理軸
3. 巨大スケールでも細かすぎる「つまむ」「正確になぞる」操作
4. 宇宙ガス・薄い大気・銀河構造を、触感として言い切る表現
5. 地球級以降でも「立つ・しゃがむ・跳ぶ」を平地感覚で使う記述

## 外部スキルから採るべき核

### 1. スケール帯で主語を変える

- 300km級までは都市・湾・平野・山脈
- 700km〜3000km級では列島・海域・雲頂帯・衛星観測欠損
- 10000km級では地球・月・潮汐・地殻・重力相互作用
- 10万km級以降では惑星・軌道・小惑星帯・恒星系
- 光年級以降では恒星間航路・銀河腕・銀河団・フィラメント・ボイド

この切替を各話の冒頭3〜5段落に必ず入れる。
序盤で読者の物差しを固定しないと、後段の破壊が全部同じ大きさに見える。

### 2. 十々瀬視点と観測側視点の落差

基本構文は以下。

- 十々瀬側: 「触ってみる」「寄せる」「並べる」「軽い」「丸い」「かわいい」
- 観測側: 「接地圧拡大」「潮位逸脱」「航路喪失」「軌道逸脱」「観測基準崩壊」

各話で最低1回は、同一事象をこの二重構造で見せる。

### 3. 破壊を「悪意」ではなく「身体の延長」にする

動詞の軸は以下に寄せる。

- 触る
- 撫でる
- ずらす
- 体重を預ける
- 近づける
- 並べる
- 回す
- 抱え込むように寄る
- 指を寄せる

結果として世界が壊れる形にする。
「壊す」が先に立つと十々瀬の軽さが弱くなる。

### 4. kmの桁を快感として使う

とくに宇宙編では、光年表記だけで済ませず、以下を数値化する。

- 身長
- 指幅
- 胸元の影の幅
- 接触域
- ずらした軌道距離
- ほどいた航路長
- 影に入った恒星数や銀河腕幅

表記の基本形は、情報を詰め込まず一行ずつ見せる。

例:

- 身長：10光年
- 約94兆6073億4725万808km
- 指幅：0.2光年
- 約1兆8921億4609万4516km
- バスト幅：0.8光年

バストや胸部曲面のように、まずスケールの印象を優先したい部位は、光年表記を単独行で見せてもよい。逆に、移動距離・軌道逸脱・影の幅など、被害や天体力学に直結する量は km を別行で添える。

## 外部スキルから修正して使う点

### 1. 雲の位置を全面補正する

外部スキル内には100km級でも「雲が腰のあたり」といった記述があるが、本シリーズでは不採用。

本シリーズの基準:

- 300km級以上: 雲は地表に貼りついた白い膜
- 700km級以上: 雲は足元の模様、雲頂輪郭、衛星画像上の白斑
- 3000km級以上: 雲は惑星表皮の薄膜、日照欠落の指標

使う描写:

- 雲頂帯が影で暗くなる
- 雲膜が足元から剥がれる
- 衛星画像の白帯が欠ける
- 対流圏が押し流される
- 雲ではなく気圧配置が乱れる

避ける描写:

- 腰で雲海を割る
- 肩に雲が触れる
- 胸元に雲が絡む
- 雲を手で丸める

### 2. 姿勢語彙をスケールに合わせて更新する

1000km級以降では、平地上の人間動作をそのまま使わない。

置換の基本:

- 跳ぶ → 荷重をずらす / 踵を浮かせる / 支える位置を変える
- しゃがむ → 球へ身を寄せる / 抱え込むように近づく
- 立つ → 地球や天体の近くへ身体を預ける
- つまむ → 指を寄せた結果、周囲ごと押しずれる
- 丸める → 模様として見えた帯を寄せる / 乱流核がまとまる

### 3. 触感の言い切りを抑える

宇宙ガス、薄い大気、銀河の網目は、以下のように見た目起点へ寄せる。

- ざらついて見える
- 粉の筋みたいに見える
- 薄膜のように貼りついて見える
- 光の濃い節に見える
- 泡の縁みたいに見える

「実際にそう触れた」と断定しすぎない。

### 4. 胸部・腹部・腰・髪・吐息・影を主役化する

本シリーズでは胸部シルエットが明確な武器なので、各話で最低1回は手指以外の破壊を前景化する。

優先ベクトル:

1. 胸元の影
2. 胸部先行接地 / 圧痕
3. 腹部・腰の荷重移動
4. 髪の掃過と観測網破壊
5. 吐息による気圧再編
6. 足先・踵の荷重変化
7. 目・反射・視界喪失の観測描写

## 章別改稿方針

### 優先A: ０５「雲より上にいる」

現状、最も外部スキルの導入効果が大きい章。

重点:

- 「雲に直接触る」発想を減らし、雲頂帯・気圧配置・衛星欠損へ変換する

### 優先B: ０４「小さな水たまりで」

- 島をつまむ場合も、指先技巧より周辺海域ごと押しずらす方向へする

重点:

### 優先C: １１〜１６ 銀河以遠

重点:

- 航路、星団、銀河腕、銀河団、フィラメント、ボイドをそれぞれ別の見え方で使い分ける
- 銀河をまだ「星の集合」程度に扱っている箇所を、泡・結び目・網目・節へ更新する
- 触感断定を避ける

改稿目標:

- 指先破壊の延長ではなく、視界と座標系そのものが壊れる段階へ移す

## 改稿ルール集

### ルール1: 1章につき1つ、主破壊ベクトルを決める

例:

- ０３: 寝返り
- ０５: 胸元の影と吐息
- ０６: 球体への接触
- ０８: 軌道のほどけ
- １２: 航路の結び目崩し
- １４: 銀河の縁なぞり

手・足・胸・髪・吐息・影を全部均等に使うより、その話の勝ち筋を一つ決めた方が強い。

### ルール2: 十々瀬の一言は軽く、記録側は重く

十々瀬の台詞は短く素朴でいい。

- かわいい
- 丸い
- 軽い
- こっちのほうが並びがいい
- ちょっとずれた
- これ、ほどける

その代わり、記録側で事象を重くする。

### ルール3: 接触前の影・圧・観測欠損を必ず挟む

巨大感は、触れた瞬間だけではなく、その前段の予兆で出る。

必須候補:

- 影が先に走る
- 雲頂帯が暗く欠ける
- 気圧差で窓や海面が先に揺れる
- 衛星画像が一部失われる
- 通信座標がずれる
- 夜空の恒星列が一部消える

### ルール4: 宇宙編では「空虚さ」を忘れない

- 惑星間距離
- 恒星間距離
- 銀河腕の疎密
- 銀河間空隙
- ボイドの大きさ

これを抜くと、全部が机の上の玩具になる。
