# 百川十々瀬 キャラクターデザインシート（三面図）

お絵描き練習用。外見メモから再構成。

---

## 基本プロポーション

| 項目 | 値 | 備考 |
|------|-----|------|
| 全頭身 | 約6頭身 | やや幼い印象を残すバランス |
| 肩幅 | 標準〜やや狭め | 胸部の横幅に対して肩は細く見える |
| 胸围(B) | 身長比で極端に大きい | 通常のグラマー体型を超え、シルエットの主役 |
| 腰(W) | 標準〜細め | 胸部との対比ですっきり見える |
| 臀(H) | 標準 | 腰回りは比較的控えめ |
| 脚 | 細め、すらり | 黒ソックスで締まって見える |

**正面からのポイント：**

- 黒衣装の正面に縦長ベージュパネルが縦のアクセント
- 胸元の金装飾が視線を集めるポイント
- 髪の房が左右へ流れ、シルエットの横幅をさらに広げる

```
   ｜   黒衣装に包まれた巨大な半球が
   ｜   胴体前面から大きくせり出す
   ｜   前方突出量 ≧ 頭の大きさ×2〜3
   ｜

   ｜── 背面に青緑ケープ
   ｜

```

**側面からのポイント：**
- 胸部の前方突出がシルエットを支配。胴体前面から大きく張り出し、重力で下方向へ垂れる質量感
- 背面には青緑のケープ/上着が見える（正面からは見えない）
- 後ろ髪が背中〜腰まで流れる
- お腹〜腰は比較的フラット（胸の下で一度シルエットが引っ込む）

## 描くときの注意点

1. **胸の質量感を逃げない** — 服の影や布の張りで立体感を出す。のっぺりした「大きく描いただけ」にしない
2. **髪のグラデーションを活かす** — 単色ではなく、房ごとに色の濃淡がある。光の当たり方で色が変わる
3. **表情は常に柔らかく** — 破壊描写との落差がこのキャラの持ち味。怒り顔・決意顔はほぼ不要
4. **ケープの存在を忘れない** — 背面・側面では必ず青緑が見える。正面でも袖や肩口から覗く
5. **サンダルの足先** — 巨大化時の足跡描写に繋がるので、つま先の形をしっかり描く
