# ０１話 300km級棚卸しと前史候補

## 1. 現在サイズ

- 身長：約300km
- この時点で雲頂13kmは身長の約4.3%
- 都市は個体ではなく灰色の染み、港湾や道路網は模様、山脈は掌紋や薄い筋として見える
- 十々瀬は「地表の玩具化」が楽しくなり始めるが、まだ地球全体そのものは玩具として扱わない

## 2. 小さく見えるもの

### 人工物
- 沿岸都市
- 港湾
- 幹線道路
- 鉄道網
- 空港
- 高圧線
- ダム

### 自然
- 湾
- 平野
- 山脈
- 海峡
- 雲頂帯
- 河口
- 丘陵

### 観測系
- 気象レーダー
- 地震計
- 衛星画像上の雲欠損
- 広域停電後の地上光

## 3. まだ玩具として面白いもの

- 都市に足跡を残すこと
- 指先で線を引くこと
- 山をつまむより、山脈や丘陵を「ちょっとした盛り上がり」として感じること
- 海で足を洗うこと
- しゃがむ、身を寄せる、体重を預けることで丸い圧痕ができること
- 夜の停電地帯の中に残る光を「星みたい」と感じること

## 4. 十々瀬の第一感想

- ちっちゃい
- 模様みたい
- 丸く残るのがかわいい
- 反応がかわいい
- 海で洗えばきれいになりそう
- 光が星みたい

※ この時点では理科的整理より、身体感覚優先の単純な感想を保つ。

## 5. 十々瀬の最初の一動作

- まずは立ち上がって世界の縮み方を確かめる
- 次に足を下ろして「自分の形が残る」ことに喜ぶ
- その後、しゃがむ・手をつく・胸元を寄せる・海で足を洗う、の順で身体の別部位を試す

## 6. 外側で最初に壊れる現象

- 窓ガラスの一斉破損
- 沿岸波の異常な先行
- 雲膜の欠損
- 地震計の急振幅
- 広域停電
- 地図上で確認できる足跡・圧痕・新湖盆

## 7. この話の主破壊ベクトル

- 主軸：足裏荷重
- 副軸：胸元の先行影 / 胸部接地圧痕 / 指先の線引き / 吐息 / 髪の掃過

## 8. 第１話改稿で強めるべき点

1. 300km級では雲を身体の中腹より上に置かない
   - 雲は足元の白い膜、衛星画像の白斑、影で暗くなる雲頂として扱う
2. 地名の細部より、湾・平野・山脈・海峡・都市圏といった地図的まとまりで見せる
3. 指だけでなく、胸元の影・接地・荷重移動を明確に前景化する
4. 十々瀬の感想は軽く、観測側の結果は重くする
5. 同じ「潰す」を繰り返さず、足跡・丸い圧痕・線状破壊・海底擦過・停電・夜景の落差で変化をつける

## 9. 300km以前の前史を後付けする候補

第１話を崩さずに前史を足すなら、「10月10日が最初の異常日」ではなく「世界地図に赤線が引かれるほど決定的に記録された最初の日」と定義するとよい。

### 候補A：数km〜10km級の前兆日
- 郊外の丘、送電線、貯水池、海辺での小規模な接触事故
- 本人は“ちょっと背が伸びた”“歩きにくい”程度
- 地方ニュース・SNS動画・断片記録にとどまり、世界史化しない

### 候補B：数十km級の非公開災害
- 山地の陥没、湾岸の潮位異常、道路網の寸断
- 政府や自治体は地震・地盤沈下・気象異常として処理を試みる
- 後から「十々瀬初期成長期の痕跡」と再分類できる

### 候補C：100km級未満の夜間行動
- 夜に海辺で足を洗う
- 髪を払って送電網を切る
- うつ伏せで休み、丸い圧痕だけが残る
- 目撃者が少なく、伝説・デマ扱いされる

## 10. 第１話への実装方針

- 冒頭に「後から見れば前兆はあったが、世界が赤線で記録したのはこの日だった」と読める含みを薄く入れる
- ただし本文の主役はあくまで300km初の決定的記録日に据える
- 前史の詳細は別話または幕間として差し込めるよう、断定しすぎない
