# 匣庭のジャステイル — シリーズ構成案

## 単位系

異世界では千倍刻みのスケール差を扱うため、以下の単位系を用いる。

## 前提

平凡な青年マイトは、ある日突然、異世界から差し伸べられた巨大な指によって家ごと摘まみ上げられ、千倍スケールの異世界に移住する。そこは「匣庭」と呼ばれる多重入れ子構造の世界——彼の千倍のメイド、百万倍の図書館管理人、十億倍の名家のお嬢様、そしてさらに上の管理者たちが階層を成して存在している。マイトは最も小さな住人として、航空機でメイドのもとへ飛び、メイドの掌に運ばれる日々を送ることになる。

## 区画管理人の修復能力

この世界では、上位スケールの存在が下位スケールの破壊を修復できる。区画管理人は専用の修復粉（光る粉状の物質）を用いて、自分の管理エリア内で発生した破壊——自身の胸による圧壊を含む——を元通りに復元する。復元された人物や構造物は、破壊前の記憶を保持したまま元の状態に戻る。この修復は完全であり、「死」は一時的な状態に過ぎない。ただし、圧壊の瞬間の記憶は残るため、住人たちにとっては軽いトラウマになる。

## 移動手段

- マイト → ミリア：専用航空機で飛行、ミリアが航空機ごと掌に乗せて運ぶ
- ミリア → パメラ：メイド専用航空機で飛行、パメラが同様に運ぶ
- パメラ → ジャステイル：図書館の蔵書復元機を用いてパメラが通常スケール（10億倍）に一時復帰、その状態でマイトを肩などに乗せてジャステイルと同席

## 蔵書復元機

図書館の中核設備。千分の一に縮小された蔵書を元のサイズに戻すための装置。ジャステイルが図書館の本を読む際に使用される。原理的には千分の一に縮小されたものなら何でも復元可能で、パメラ自身もこの装置で本来の10億倍スケールに一時的に戻ることができる。ただし長時間の使用はできず、図書館管理を離れられるのは「お茶一杯分」程度。

## 縮小設定

ミリアとパメラは本来、ジャステイルと同じくマイトの10億倍（約170万km）の体格を持つ。しかしそれぞれの役割——ミリアはマイトの直接世話、パメラは1/1000スケールの図書館管理——を遂行するため、自らのスケールを縮小している。
- ミリア：10億倍 → 1,000倍（100万分の1に縮小）
- パメラ：10億倍 → 1,000,000倍（1,000分の1に縮小）

縮小は任意で行われており、二人とも本来のスケールに戻ることも可能。ただし役割上、常時縮小状態で生活している。ジャステイルとは縮小前からの知己であり、主従というよりは役割分担に近い関係。

## 館の真実

匣庭は現在、ミリアの「原初の力」によって本来より遥かに小さなスケールに縮小されている。これは封印ではなく、ジャステイルの母が遺した「いずれ匣を開くときまで、館を守れ」という意志に基づくもの。ミリアは幼い頃からその力を受け継ぎ、自覚なく館全体を縮小し続けている。ラズリーネはジャステイルの母にも仕えた古参であり、この真実を知る唯一の人物。

ジャステイルの母は、途轍もなく巨大な存在だった。現在のジャステイルの10億倍スケールですら「異世界では通常サイズ」とされていることに、ジャステイルは幼い頃から違和感を持ってきた。ラズリーネから「お母様はとても大きかった」と教えられて育ったものの、詳しいことは明かされず、今日まで過ごしてきた。

ジャステイルの乳腺組織が「共鳴体」としての性質を持つのは、母から受け継いだもの。これがミリアの原初の力と共鳴することで、館の本来のスケールへの回帰——すなわち「匣を開く」——が可能になる。

## 天体の有無

この異世界において、星・星団・星間物質などの天文学的構造は、大区画管理者（10^18倍）のオフィスより外側のスケールにのみ存在する。大区画管理者のオフィスの外には宇宙に類似した空間が広がっており、宇宙管理人（10^21倍）はそこを散歩し、星団を蹴飛ばしたり整えたりできる。

それよりミクロのスケール——ジャステイルの部屋、図書館、ミリアの部屋を含む——には天体は存在しない。部屋の光源は壁面や天井の自発光による。作中では天体の不在を明示的に説明せず、単に星や宇宙空間を描写しないことで表現する。

## ジャステイルの衣装と作業姿勢

ジャステイルのB値は表示時6.32 Ra、本来時12.6 Raあり、通常の衣服では両胸をまとめて覆うことが物理的に不可能に近い。そのため、左右の乳房それぞれに独立した布（乳房別の衣）を着用している。布は各乳房の曲面を包み込むように巻かれ、胸の谷間——左右の乳房のあいだ——は開放されている。

机作業時は、左右の乳房を床に下ろし、そのあいだの空間に机を設置する。机は胸の谷間の横幅に合わせた細長い特注品で、両肘で乳房を軽く外側に押さえることで中央の作業スペースを確保する。左右の乳房はそれぞれ床の上で布に包まれた巨大な丘となり、呼吸に合わせてゆっくりと揺れて作業スペースの幅を伸縮させる。伸びをする際は肘の押さえが外れ、左右の乳房が前方で互いに接近し、部屋の光を遮る。前方から見ると、二つの巨大な丘に挟まれた峡谷の底に机と彼女の顔が覗く構図になる。

この姿勢により、胸が机を完全に覆って作業不能になる問題を回避している。

## 全体構想（三幕構成）

### 第一幕「匣のなか」（1〜7話／済）
マイト到着、世界の仕組み把握、住人との関係構築、ドットルリルの圧壊と修復。

### 第二幕「匣を開く」（8〜16話／済）
- ラズリーネ、ティア、ネイラ、ノーラ初登場（第8話）
- ラズリーネがマイトを運び、ジャステイルと至近距離で対面（第9話）
- パメラの蔵書調査：母の直筆記録発見（第10話）
- ミルカ初登場（第11話）
- ウルスラ来訪→宇宙管理人打診（第12話）
- アウレリアのオフィス＝ドットルリル懲罰回（第13話）
- 館の大改装（第14話）
- ジャステイル惑星リハーサル（第15話）
- ミリアの力が共鳴し始める（第16話）

### 第三幕「匣の外」（17〜19話／完結）
- ウルスラ正式指示、ミリア臨界→封印解除→館ごと巨大化、マイト地球帰還（第17話「匣、開く」）
- ジャスティがマイトの宇宙を乳腺に収容、マイトが夜空でその光景を目撃（第18話）
- 五日後——空が裂け、セプチの手が差し伸べられる。マイト、匣庭へ再訪（第19話）
