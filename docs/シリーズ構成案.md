# 匣の館 — シリーズ構成案

## 前提

平凡な青年ユウトは、ある日突然、異世界から差し伸べられた巨大な指によって家ごと摘まみ上げられ、千倍スケールの異世界に移住する。そこは「匣の館」と呼ばれる多重入れ子構造の世界——彼の千倍のメイド、百万倍の図書館管理人、十億倍の名家のお嬢様、そしてさらに上の管理者たちが階層を成して存在している。ユウトは最も小さな住人として、航空機でメイドのもとへ飛び、メイドの掌に運ばれる日々を送ることになる。

## 物理的入れ子構造

```
超管理職の世界
└── 大区画管理者のオフィス
    └── 区画管理者の机の上の管理エリア
        └── セレナ（お嬢様）の部屋
            └── 棚の一区画：図書館（ルーチェ管理）
                └── メイド（ミリア）の部屋
                    └── ユウトの家
```

## 移動手段

- ユウト → ミリア：専用航空機で飛行、ミリアが航空機ごと掌に乗せて運ぶ
- ミリア → ルーチェ：メイド専用航空機で飛行、ルーチェが同様に運ぶ
