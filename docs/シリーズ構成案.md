# 50kmの妻リカと夫の日常災害録 — シリーズ構成案

## 概要

あらゆるサイズの人間が共存する世界。身長50kmの「重度変動型巨人族」リカ（26歳）と、通常サイズの夫ケンジ（28歳・会社員）の日常。リカのうっかりが引き起こす災害と、それでも彼女を支える夫の姿を描く。

日常コメディから始まり、後半ではリカの巨大化能力が暴走し惑星級・銀河級へとエスカレートする。

## 世界設定

- 巨人族と通常サイズの人間が共存する現代社会
- インフラは全サイズ対応済み（道路レーン、商業施設、宿泊施設）
- リカは「変動型巨人族」——基準身長50kmから上方向にのみサイズ可変
- 年1回の巨大化健診が法律で義務付け
