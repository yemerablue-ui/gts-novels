# GTS Scale Descriptions

巨大娘（GTS）小説執筆のためのスケール別描写リファレンス集。

## いつ使うか

以下のような場面で `scales/` 配下の該当ファイルを **Read** で読み込んでから執筆する：

- 既存のGTS小説に新しい巨大化スケールの場面を追加する
- 巨大化展開（成長していくシーン）で段階的に表現を変える
- 破壊描写・周囲反応・物理現象（気象、重力、音響）の語彙を増やしたい
- スケールに合った具体的な対比物（建物・地形・天体）を探したい

## 使い方の基本

### 1. スケールを決める
場面の身長を決め、最も近い段階のファイルを選ぶ。段階間の身長（例：50km）の場合は前後二段階（10km と 100km）を読み込んで、両者の表現を線形補間する感覚で書く。

### 2. 必ず含める要素
各ファイルは以下のセクションを持つ。場面に合わせて取捨選択：

- **身体感覚**：身長に対する身体の各部位の感覚
- **見下ろし視点**：巨大娘から下を見たときの認識
- **見上げ視点**：地上の観察者からの認識
- **物理現象**：重力・気象・音響・光学の影響
- **破壊パターン**：何がどう壊れるか
- **語彙集**：使える単語・表現
- **対比物リスト**：具体的な建物・地形・天体

### 3. 「上下視点の落差」を必ず使う
GTS文芸の核は、巨大娘の認識と地上民の認識の **致命的な落差**。同じ事象でも二つの視点で書くことで読者の脳内スケールが定着する。

例：100m級
- 巨大娘視点：「足を一歩前に出した」
- 地上視点：「直径六十メートルの肉色の柱が、空から振り下ろされた」

### 4. トーンをキャラクターに合わせる
破壊描写の重さは、物語のトーンと巨大娘のキャラクター性に合わせて調整する。コメディ全振りなら慌てぶり・申し訳なさ・周囲のツッコミを並走させ、シリアスならスケール感と感情の重みを前面に出す。各スケールファイルの「トーン調整のコツ」セクションを参照。

### 5. 過剰なエロは禁止
胸の描写はスケールの偉大さ・物理的存在感を伝える方向に集中。性的行為・性器への言及は規約違反となるため不可。

## 作風の参考

本スキルは以下のGTS文芸作風を参考に設計：

- **十六夜（hearthiel）**：人物心理を丁寧に描きつつスケール感を保持する書き方
- **唐変木**：宇宙〜超宇宙スケールでの認識崩壊を文章で表現する書き方
- **こおろぎ「1/1じゃない艦娘シリーズ」**：軍事・海事の対比物を使った大きさの実感

ただし直接の引用はせず、表現規範のみを踏襲する。

## ファイル間のリンク

スケール上昇・下降を含む場面では、複数ファイルを順番に Read して、段階ごとの語彙を切り替える。例えば「100m → 1km → 10km」と成長する場面では三ファイルを順次参照すること。
