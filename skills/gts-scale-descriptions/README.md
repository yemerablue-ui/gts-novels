# GTS Scale Descriptions スキル

巨大娘（GTS）小説執筆のためのスケール別描写リファレンス集。  
10m 級から宇宙越えサイズまで **11 段階**のガイドラインファイルを提供する。

---

## インストール

### ユーザーレベル（全エージェント共有・推奨）

```
~/.claude/skills/gts-scale-descriptions/
  SKILL.md
  README.md
  MANIFEST.md
  scales/
    10m.md
    100m.md
    1km.md
    10km.md
    100km.md
    1000km.md
    10000km.md
    100000km.md
    solar-system.md
    galactic.md
    beyond-universe.md
```

ディレクトリをそのまま `~/.claude/skills/` にコピーすればすべてのエージェントから利用可能。

### プロジェクトレベル（特定プロジェクトのみ）

プロジェクトルートの `.claude/skills/` 以下に配置する。

---

## 使い方

このスキルは **読み込み専用リファレンス** として機能する。  
執筆時に Claude に次のように伝えると自動でガイドラインが適用される：

```
「○km 級の巨大化シーンを書いて」
「破壊描写の語彙を増やしたい」
「10km から 100km に成長する描写を段階的に」
```

### 手動で参照ファイルを指定する場合

```
scales/10m.md        — 10m 級（民家・電柱）
scales/100m.md       — 100m 級（東京タワー比）
scales/1km.md        — 1km 級（ブルジュ・ハリファ比）
scales/10km.md       — 10km 級（成層圏・エベレスト比）
scales/100km.md      — 100km 級（カーマンライン比）
scales/1000km.md     — 1000km 級（月の直径比）
scales/10000km.md    — 1万km 級（地球直径比）
scales/100000km.md   — 10万km 級（木星直径比）
scales/solar-system.md  — 太陽系サイズ（〜90億km）
scales/galactic.md      — 銀河サイズ（10万光年）
scales/beyond-universe.md — 宇宙越えサイズ（930億光年超）
```

## 各スケールファイルの構成

| セクション | 内容 |
|----------|------|
| 身体感覚 | 身長に対する各部位の感覚・具体的寸法 |
| 見下ろし視点 | 巨大娘が下を見たときの主観描写 |
| 見上げ視点 | 地上の観察者・人類から見た描写 |
| 物理現象 | 重力・気象・音響・光学・時空への影響 |
| 破壊パターン | 接触物の壊れ方・事故の種類 |
| 語彙集 | 身体描写・動作描写・音・感情の用語集 |
| 対比物リスト | 建物・地形・天体の具体的数値 |
| トーン調整のコツ | コメディ/シリアス/クライマックスの処理方針 |

---

## 対応トーン

- **コメディ**：ドジっ子・申し訳なさ・周囲のツッコミを重視
- **シリアス**：スケール感と感情の重みを前面に出す
- **クライマックス/感動**：宇宙越えスケールの孤独感・帰還の物語

各スケールファイルの「トーン調整のコツ」セクションに詳細あり。

## 規約上の注意

- 胸の描写はスケールの物理的存在感の表現として使用する
- 明確な性的行為・性器への言及は規約違反のため不可
