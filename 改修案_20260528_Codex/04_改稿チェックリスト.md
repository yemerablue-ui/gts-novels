# 改稿チェックリスト

## 全話共通

- [ ] 十々瀬の巨大化原因を説明しすぎていない
- [ ] 十々瀬が悪意ある支配者になっていない
- [ ] 十々瀬が幼すぎる性格になっていない
- [ ] かわいさを狙った描写が過剰になっていない
- [ ] 十々瀬の軽い台詞や認識が、本人の尺度として効いている
- [ ] 観測側の通信・報告が多すぎない
- [ ] 個人単位の残酷描写へ寄りすぎていない
- [ ] その話の中心画面が一つ言える
- [ ] 前後の話と同じ破壊パターンが続きすぎていない
- [ ] 雲・地形・天体・銀河のスケールに違和感がない
- [ ] 胸元、足、指、髪、吐息などの役割がスケールに応じて変わっている
- [ ] 数字の桁がスケール快感として機能している
- [ ] 詩的な比喩が多すぎず、何が起きているか明瞭に読める

## 地上編 01〜05

- [ ] 雲を巨人の肩や胸に絡ませすぎていない
- [ ] 都市や道路を、十々瀬の視界では模様・線・染みとして扱えている
- [ ] 海、湾、海峡、波紋、泡でスケールが出ている
- [ ] 地形改変の記録名が印象に残る

## 地球・惑星編 06〜10

- [ ] 地球の丸みを、平面から球への認識転換として描けている
- [ ] 月、火星、木星、太陽がそれぞれ違う質感で見えている
- [ ] 軌道、影、大気、衛星、リングなど、惑星編らしい語彙がある
- [ ] 艦隊は十々瀬には小さな光として見えている

## 銀河・宇宙編 11〜16

- [ ] 艦隊や航路が個別描写ではなく、線・点・網として扱われている
- [ ] 星間距離、航路、座標、参照点の破綻が描けている
- [ ] 銀河、銀河団、フィラメント、宇宙網目の絵面が話ごとに違う
- [ ] 最終話に向けて派手さだけでなく静けさが増している
- [ ] 宇宙を説明しきらず、外側へ出ていく余白が残っている
- [ ] 宇宙論・観測用語が多すぎず、十々瀬の視点と地の文が主軸になっている

## 地雷系ピンク衣装を採用する場合

- [ ] 初出で衣装の印象が短く伝わる
- [ ] 毎話で同じ服装説明を繰り返していない
- [ ] 衣装ディテールがミクロ視点やスケール比較に接続している
- [ ] 衣装ディテールが巨大感の物差しとして機能している
- [ ] 黒レース、チェーン、ハート金具などがスケール描写へ自然に混ざっている
- [ ] 宇宙編で航路・フィラメント・レース・チェーンの対応が効いている
- [ ] 衣装描写が単なる服飾カタログではなく、視点・スケール・影・接触・比較対象に結びついている
