# 2026-05-28 Codex改修案

DeepSeek改稿ラインや既存レビューとは別枠で作った、Codex側の独立した改修メモ。
本文ファイルは直接変更せず、次に改稿・レビュー・挿絵組み込みを行うときの作業台として使う。

## このフォルダの目的

- 現行16話を大きく壊さず、次の磨き込みで効きそうな観点を整理する
- DeepSeekへ投げるときの指示文を、章ごとに迷わず作れるようにする
- 新しい地雷系ピンク衣装と挿絵群を、本文演出へ自然に接続する
- 重複しがちな破壊パターンを分散し、各話の絵面と読後感を分ける

## ファイル構成

- `00_全体改修方針.md`
  - シリーズ全体で優先したい改修軸。
- `01_章別改修案.md`
  - 1話から16話までの章別メモ。強める点、避ける点、追加候補。
- `02_衣装と挿絵の活用案.md`
  - 地雷系ピンク衣装を本文へ馴染ませる案と、生成済み挿絵の使いどころ。
- `03_DeepSeek投入用プロンプト.md`
  - DeepSeekへ渡すための汎用プロンプト、章別プロンプト、チェックリスト。
- `04_改稿チェックリスト.md`
  - 改稿後に見る短い点検リスト。

## 使い方

1. まず `00_全体改修方針.md` で今回の改修思想を確認する。
2. 対象話の作業前に `01_章別改修案.md` を読む。
3. DeepSeekへ投げる場合は `03_DeepSeek投入用プロンプト.md` から該当部分を使う。
4. 挿絵を本文に入れる場合は `02_衣装と挿絵の活用案.md` を参照する。
5. 改稿後に `04_改稿チェックリスト.md` で最終確認する。

## 今回の前提

- 十々瀬の原因説明は増やさない。
- 十々瀬の倫理観は大きく変えない。悪意ではなく、好奇心と遊びの延長。
- 人類側・航宙軍側の固定主人公は置かない。
- 記録文学風の硬さは残すが、各話の絵面と感情の流れを少し読みやすくする。
- 新衣装は「地雷系ピンク寄せ」を正式候補として扱う。衣装ディテールは巨大感の物差しとして積極的に使い、避けるのはスケールや視点に接続しない服飾カタログ化だけにする。
