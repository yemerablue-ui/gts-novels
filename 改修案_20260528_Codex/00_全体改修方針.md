# 全体改修方針

## まず残すもの

現行稿の強みはかなりはっきりしている。

- 十々瀬の軽い台詞と、外部記録の冷たい巨大災害描写の落差
- サイズが都市、地形、気象、惑星、恒星系、銀河、宇宙へ進む快感
- 原因説明を捨てて、結果だけが淡々と巨大化していく潔さ
- 足跡、指跡、胸元の影、髪、吐息、軌道、航路、銀河構造など、身体と世界の接触面が多いこと

改修では、ここを薄めない。特に十々瀬を反省させすぎたり、敵役や固定視点を主人公化したりすると、このシリーズの味が鈍る。

## ユーザー所感を受けた補正

2026-05-28時点の試作稿への所感:

- 詩的な表現はあまり求めていない
- 十々瀬の性格が幼すぎる方向へ寄るのは違う
- かわいさ自体も主目的ではない
- 観測側の表現は必須ではなく、彼女の視点の文章と地の文だけでもよい

以後のCodex改修案では、以下へ寄せる。

- 比喩を増やすより、身体感覚、スケール差、結果としての地形・天体変化を明瞭に書く
- 十々瀬は幼い子どもではなく、軽く、悪びれず、好奇心のまま動く人物として扱う
- 「かわいい」を狙って足すのではなく、本人の軽さと規模の落差を残す
- 観測側の通信・報告は、必要な箇所だけ短く使う。必須要素にはしない
- 文章の軸は、十々瀬の見え方と、三人称地の文による現象描写に置く

## 今回の改修テーマ

### 1. 各話の「主役となる身体部位・現象」を一つ決める

現行稿は豊富な破壊が入っている反面、話によっては指、胸元、影、髪、足が一度に出てきて、読後にどの絵が残るか散りやすい。

各話に一つだけ、読者が覚える主役を設定する。

- 1話: 手を見る、膝と海、初めての影
- 2話: 指先と首都圏
- 3話: 寝返りと接地帯
- 4話: 足と海岸線
- 5話: 伸びと雲膜
- 6話: 地球の丸みと片足
- 7話: 月を小石として見る掌
- 8話: 火星の軌道をほどく指
- 9話: 木星圏の小さな光
- 10話: 恒星系の玩具箱
- 11話: 防衛線をまたぐ指先
- 12話: 航路を糸としてつまむ指
- 13話: 銀河腕をかき混ぜる手
- 14話: 銀河の縁をなぞる指
- 15話: 宇宙網目をたぐる指と髪
- 16話: 宇宙全景を見下ろす視線と影

脇の破壊は残してよいが、章の中心画面を一枚に絞る。

### 2. 十々瀬の主観を「本人の尺度」として増やす

記録側の描写は十分強い。次の改修では、各話に一つだけ十々瀬の尺度で見た呼び方や認識を置くと、彼女の軽さと災害規模の落差が増える。

ただし、幼さやかわいさを強調するための言い換えにはしない。十々瀬は世界を小さく見ているだけで、幼児的に理解できていないわけではない。

例:

- 都市: 模様、線、濃淡、細かい凹凸
- 高速道路: 細い糸、ひび、飾り線
- 雲: 白い膜、粉、綿ではなく地図の擦れ
- 木星の縞: 布柄、飴玉の模様
- 艦隊: 小さな光、線香花火、ビーズ
- 航路: 光の線、飾り紐、規則的な列
- 銀河腕: 砂場、光る粉
- 宇宙網目: 網目、糸、細い構造

地雷系衣装を採用する場合、黒レース・チェーン・ハート金具の語彙と、航路・フィラメント・銀河団の語彙を軽く響かせると後半が映える。ただし、衣装比喩を詩的に引っ張りすぎない。

### 3. 観測側は補助に留める

観測側の通信や報告は、必須ではない。スケールを示す補助や、十々瀬の主観では見えない結果を短く置くために使う。

主軸は、十々瀬の見え方と、通常の地の文による現象描写に置く。観測側の台詞が連続して、軍事レビューや報告書が主役にならないようにする。

改修方針:

- 地上編: 地形・気象・インフラの変化を具体的にする
- 惑星編: 軌道、影、大気、リング、衛星の再配置を中心にする
- 恒星系編: 艦隊は光点列として扱い、個別戦闘は短くする
- 銀河編: 星団、星間雲、航路、銀河腕の変形を中心にする
- 宇宙編: 宇宙網目そのものの変形を中心にする

観測不能や座標破綻は、必要な箇所にだけ入れる。

### 4. 「胸元の存在感」と衣装ディテールを毎話同じ言い方にしない

胸元の巨大さは重要だが、毎話で同じように「影」「接地」「圧痕」と書くと単調になる。スケールに応じて役割を変える。

- 1〜4話: 地表・海面に対する面状接触、影、圧痕
- 5〜6話: 大陸規模の日照欠落、気圧境界、地球曲面への遮蔽
- 7〜10話: 惑星への影、衛星軌道への重力的影響、太陽光遮蔽
- 11〜13話: 恒星間航路・艦隊列の遮蔽、星間塵の押し分け
- 14〜16話: 銀河・銀河団・宇宙背景光を覆う巨大な幕

本文では「胸」とだけ書くより、衣装ごとの見え方を混ぜる。

- 黒レースに縁取られた桃色の巨大な曲面
- ハート金具の影が都市圏より大きく落ちる
- チェーン装飾の弧が、地上からは人工衛星の軌道線のように見える
- 宇宙編では、淡いピンクの曲面が星雲を背景にひとつの天体の縁のように見える

ただし衣装描写を性的に寄せすぎず、巨大な衣装面・影・遮蔽・装飾のスケールで表現する。

衣装説明は、それ自体を避ける必要はない。巨大娘小説では、ミクロ視点から衣装の縫い目、レース、金具、靴底、布の曲面を描くことで、読者に巨大感を実感させられる。むしろ小さい側が衣装の一部に圧倒される描写は、身体サイズを直接説明するより強い物差しになる。

使うべき衣装描写:

- 地上・艦隊・星系など、彼女より小さい視点から見た衣装の巨大さ
- レースの縁が地形や星雲に落とす影
- ハート金具やチェーン飾りが、都市・艦隊・星団より大きく見える対比
- 靴底の溝、スカートの裾、袖口、リボンが、地形や宇宙構造へ与える影響

避けたい衣装描写:

- スケールや視点に接続しない、単なる服飾カタログ
- 本筋の動作、影、接触、比較対象と無関係な説明が長く続くこと
- 毎話同じ順番で服装を説明し直すこと

### 5. 章末の引きを少し抑える

一挙公開前提なら、各話末尾で強く「次はあれ」と締めすぎるより、記録が自然に次のスケールへ滑る方が読みやすい。

おすすめの章末:

- 十々瀬の軽い一言
- 観測値の更新
- 記録名だけが残る
- 次の対象が視界に入る

この四つのうち二つまでに絞る。

### 6. 詩的な比喩を増やしすぎない

試作稿では比喩と装飾語がやや前に出た。今後は次の順序にする。

1. 何をしているか
2. 彼女にはどう見えているか
3. 実際に何が変わったか
4. 必要なら短い比喩

比喩は一段落に一つまでを目安にする。特に宇宙編では、きれいな表現より「どの構造が、どう動いたか」を優先する。

## 改修優先順位

1. 重複する破壊文型を減らす
2. 各話の中心画面を一つにする
3. 十々瀬の尺度で見た認識を足す
4. 観測側の描写を必要箇所だけに絞る
5. 地雷系ピンク衣装を、必要な場面だけ本文へ馴染ませる
6. 詩的・装飾的な表現を減らし、現象の明瞭さを優先する

## 今回は避けること

- 巨大化の原因を説明する
- 人類側や航宙軍側に固定主人公を立てる
- 十々瀬に本格的な罪悪感や贖罪を持たせる
- 十々瀬を幼く、かわいさ狙いに寄せすぎる
- 各話を独立短編のようにきっちり締めすぎる
- 破壊被害を個人単位で長く描く
- 衣装説明を、スケール感、ミクロ視点、影、接触、比較対象に接続しないまま服飾カタログ化する
- 観測側の通信や軍事報告を必須パーツ化する
- 詩的な比喩を重ねて、何が起きているかを曖昧にする
