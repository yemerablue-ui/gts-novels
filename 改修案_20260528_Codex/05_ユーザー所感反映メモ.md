# ユーザー所感反映メモ

作成日: 2026-05-28

## 受けた所感

- あまり詩的な表現は求めていない
- 試作稿では十々瀬の性格が幼すぎる
- かわいさも主目的ではない
- 観測側の表現は必須ではない
- 彼女の視点の文章と地の文だけあればよい
- ただし、悪いと言えるほどではない

## 解釈

今回のCodex案は、以下へ寄せすぎた。

- かわいい誤認
- 詩的・装飾的な宇宙比喩
- 観測側の通信による硬い対比
- 地雷系衣装のモチーフを文章へ強めに反映する方向

これらは完全に不要ではないが、主軸に置くべきではない。

## 今後の修正方向

### 1. 詩的表現を減らす

比喩は使うが、文章の主役にしない。

優先順:

1. 十々瀬が何を見ているか
2. 何をしたか
3. 何が変わったか
4. 必要なら短い比喩

避ける方向:

- 宇宙を「刺繍」「レース」「宝石」などへ何度も言い換える
- 衣装装飾と宇宙構造を過剰に対応させる
- きれいな文のために、現象が曖昧になる

### 2. 十々瀬を幼くしない

十々瀬は無邪気ではあるが、幼児的ではない。

目指す人物像:

- 悪びれない
- 好奇心で動く
- 反応が軽い
- 世界の小ささを当然のように受け入れる
- 破壊を深刻に受け止めない

避ける人物像:

- 何でも「かわいい」と言うだけ
- 幼い比喩ばかり使う
- 理解力が低そうに見える
- 語尾やリアクションが子どもっぽすぎる

### 3. 観測側を減らす

観測側は、必要なときだけ使う補助。

使ってよい場面:

- 十々瀬の視点だけでは結果が伝わらないとき
- スケール差を一行で締めたいとき
- 宇宙軍の存在を最低限示したいとき

減らす場面:

- 通信台詞が連続する場面
- 兵器説明が長くなる場面
- 観測用語が本文の流れを止める場面

### 4. 第13話を次に直すなら

現在の試作稿からさらに直すなら、以下。

- 冒頭の衣装説明は、単なる服飾紹介ではなく銀河スケールの視点差へ接続する
- 「黒レース」「ハート金具」は、星間雲、艦隊、航路、銀河腕との比較に効く箇所で使う
- 軍通信を半分程度に削る
- 「かわいい」「きらきら」系の台詞を減らし、より素の反応にする
- 銀河腕の変形を、詩的比喩より位置・密度・流れの変化として書く

ただし、衣装説明そのものを避ける必要はない。
巨大娘ものでは、小さい側から見た衣装ディテールが巨大感の物差しになる。

有効な衣装描写:

- 黒レースの縁が星間雲を遮る
- ハート金具の反射が艦隊の光点より大きい
- チェーン飾りの弧が航路光と同じ画面に入る
- 袖口や裾の縫い目が、銀河腕の暗黒帯と重なる

避けたいのは、スケールと無関係な服装説明が本文の流れを止めること。逆に、衣装ディテールが小さい側を圧倒する構図を作るなら、説明量があっても巨大娘小説として有効に働く。

例:

```text
「砂場みたい」
```

これは残してよい。

```text
「ここ、混ぜたらもっときれいかも」
```

これは少し幼い／かわいさ寄りなので、

```text
「こっちへ流したら、どうなるんだろ」
```

のように、好奇心寄りへ変えるとよい。

## 新しい基準

Codex側の改修は、以後「かわいくする」ではなく「十々瀬の尺度で淡々と巨大にする」方向を優先する。

衣装ディテールは、スケール差を出せるなら積極的に使ってよい。

## 追加所感: 衣装説明は悪ではない

衣装説明そのものを削る方針にはしない。
巨大娘小説では、ミクロ視点の衣装ディテールが巨大感を際立たせる。黒レースの一山、ハート金具の反射、チェーンの一コマ、厚底靴の溝、布の曲面が、都市・艦隊・星雲・航路より大きく見えること自体が見せ場になる。

判断基準は量ではなく、巨大感、視点、接触、影、比較対象に接続しているかどうか。
