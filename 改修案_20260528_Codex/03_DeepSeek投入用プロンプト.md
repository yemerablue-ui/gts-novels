# DeepSeek投入用プロンプト

## 汎用プロンプト

以下を対象話ごとに貼り替えて使う。

```text
以下の小説本文を、既存の文体と設定を保ったまま改稿してください。

目的:
- 百川十々瀬本人の尺度で見た世界の小ささと、実際に起きる地形・天体規模の変化の落差を強める
- 巨大化原因の説明は追加しない
- 固定視点人物は増やさない
- 記録文学風の硬さは必要に応じて残しつつ、小説としての流れを読みやすくする
- 詩的な表現や装飾的な比喩を増やしすぎない
- 百川十々瀬を幼くしすぎない。かわいさを狙って足すより、軽さ・好奇心・悪びれなさを保つ
- 観測側の通信や報告は必須ではない。必要な箇所だけ短く使う
- 露骨な性的描写や個人単位の残酷描写は増やさず、巨大な身体・影・接触面・地形変化・天体構造の変化として描く

今回の章で特に強めたい中心画面:
（ここに章別改修案の「中心画面」を貼る）

今回足してよい要素:
（ここに章別改修案の「追加候補」を貼る）

今回避けたいこと:
（ここに章別改修案の「避けたい点」を貼る）

出力形式:
- 改稿本文のみ
- 解説や箇条書きは不要
- 本文冒頭に章タイトルは追加しない
```

## レビュー専用プロンプト

```text
以下の小説本文をレビューしてください。

レビュー観点:
1. この話の中心画面が一つに絞れているか
2. 十々瀬本人の尺度で見た世界と、実際に起きる変化の落差が出ているか
3. 破壊パターンが前後の話と重複しすぎていないか
4. サイズに対して、雲・地形・惑星・恒星・銀河の扱いに違和感がないか
5. 胸元・足・指・髪・吐息などの身体描写が、単なる反復でなくスケールに応じた役割を持っているか
6. 表現が過度に説明的、または過度に抽象的になっていないか
7. 詩的な比喩やかわいさ狙いが過剰になっていないか
8. 観測側の通信・報告が多すぎて、十々瀬の視点と地の文を邪魔していないか

出力形式:
- 重要度順の指摘
- 修正方針
- 具体的に差し替えたい文の例
```

## 章別の短縮指示

### 01

中心: 海で目覚め、手と足元の世界を見比べる。

指示:

```text
第一話は初登場回なので、十々瀬のかわいい驚きと、海・陸・雲が小さく見える感覚を強めてください。破壊行為を増やすより、膝をついた海、手の影、足元の都市が模様に見える落差を優先してください。
```

### 02

中心: 首都圏を模様として見下ろし、指でなぞる。

```text
首都圏の道路・鉄道・湾岸・川を、十々瀬には薄い模様や編み目として見せてください。都市破壊の結果だけでなく、触れる前の指先と都市のスケール差を強めてください。
```

### 03

中心: 寝返り。

```text
寝返りという普通の動作が地方規模の地形改変になる落差を強めてください。接地部位の説明を増やしすぎず、眠そうな十々瀬の感覚を中心にしてください。
```

### 04

中心: 海を水たまりのように扱う。

```text
1000km級として、雲は身体に絡ませず地表の薄い白膜として扱ってください。海峡、湾、島、港湾が靴底や足の動きで変わる描写を中心にしてください。
```

### 05

中心: 雲より上で伸びをする。

```text
雲を地表へ貼りついた薄膜として扱い、十々瀬の伸び・笑い・吐息だけで気象が変わる描写を強めてください。観測側は全身ではなく雲頂欠損や日照変化から姿勢を推定する形にしてください。
```

### 06

中心: 地球の丸み。

```text
地球を地面ではなく球として認識する転換を強めてください。壊れる結果より、「地球って丸いんだ」という発見と、足先や掌で球面へ触れる感覚を中心にしてください。
```

### 07

中心: 月が小石に見える。

```text
月を白い小石、地球残骸を静かな輪として扱う宇宙の静けさを強めてください。悲劇を長く描くより、十々瀬の軽い好奇心と、軌道がほどける記録を対比してください。
```

### 08

中心: 火星と軌道。

```text
火星そのものの破壊だけでなく、触れる前から影や重力で軌道がほどける異常さを強めてください。赤い砂、薄い大気、小惑星帯の弧を印象的に使ってください。
```

### 09

中心: 木星圏の小さな光。

```text
木星の縞模様と、異星防衛網の規則的な光点列を中心にしてください。十々瀬は攻撃ではなく光る列に興味を持つように描き、艦隊は個別より隊列・点滅・同期で表現してください。
```

### 10

中心: 恒星系の玩具箱。

```text
太陽系全体を玩具箱として見せてください。惑星の軌道は光る輪、艦隊は線香花火、太陽は明るいビー玉として、十々瀬の視界で統一してください。
```

### 11

中心: 銀河の防衛線。

```text
恒星間の防衛線を、星と星の間に張られた小さな光の線として見せてください。三光年級の十々瀬の指先が防衛線をまたぐことで、距離計や座標が破綻する描写を強めてください。
```

### 12

中心: 航路を飾り紐としてほどく。

```text
星間航路をイルミネーションや飾り紐のように見せ、十々瀬が結び目をつまんでほどく場面を中心にしてください。結果は航路座標の全面不一致として冷たく記録してください。
```

### 13

中心: 銀河腕をかき混ぜる。

```text
銀河腕を光る砂場、星間雲を薄い綿、星団を光る砂として扱ってください。ただし詩的な比喩を増やしすぎず、十々瀬が何をして、銀河腕がどう変形したかを明瞭に書いてください。主力艦隊の戦闘は必要最小限にし、十々瀬の指が星間ガスをかき混ぜる絵面を中心にしてください。
```

### 14

中心: 銀河を手元の渦模様として見る。

```text
第十三話と差別化し、今回はかき混ぜより観察に寄せてください。銀河を手元の渦模様、周囲の小銀河を泡として見下ろす静かな場面を中心にしてください。
```

### 15

中心: 宇宙網目をたぐる。

```text
宇宙大規模構造のフィラメントを、黒い布についた糸くずやレースのように扱ってください。観測側は彼女自身ではなく、宇宙地図の変形を追う形にしてください。
```

### 16

中心: 宇宙全景。

```text
最終話は派手な破壊より、宇宙全体が淡い網目模様として見える静かな到達感を重視してください。文明の信号は届かないのではなく、尺度が合わず受け取れないものとして描いてください。
```
